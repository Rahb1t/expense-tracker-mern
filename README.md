# Expense Tracker (MERN)

> Full stack expense tracker

## Usage

```
change config.env file in config folder
```

```
 npm install
 cd client npm install
 cd ..

 # Run front and backend
 cd server
 npm run dev

 # Backend only
 cd server
 npm run server

 # Frontend only
 cd client
 npm run client

 # Build client
 cd client
 npm run build

 # Prod
 npm start
```
